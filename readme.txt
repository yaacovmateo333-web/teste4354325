== Getting Started ==
 
To get a social network on your WordPress site up and running, follow these steps:
 
* Login to your WordPress dashboard.
* Choose "Plugins".
* Click "Add new".
* Search for "PeepSo".
* Click "Install".
* Go to "Appearance -> Menus"
* Select the menu you'd like to link to your social network.
* Add the page "Recent Activity" to that menu
* Save.
 
Your new **social network** will now be displayed on your site.

== Installation ==

= From your WordPress dashboard =

1. Visit "Plugins > Add New"
2. Search for "PeepSo"
3. Press "Install"
4. Activate PeepSo from your Plugins page.


= From WordPress.org =

1. Download PeepSo
2. Navigate to the Admin area of your site.
3. Go to Plugins > "Add New".
4. Select "Upload Plugin".
5. Select the ZIP installation file of PeepSo.
6. Activate PeepSo from your Plugins page.


== Frequently Asked Questions ==

= Is PeepSo GDPR Compliant? =

PeepSo does have the options and configuration settings that will help you as a responsible site administrator to meet the obligations and rights enacted under the GDPR. Please note that the built-in configuration options for the GDPR cover only PeepSo plugins. Should you use any other 3rd party software that also needs to be GDPR compliant, it needs to be covered by its respective developers. PeepSo does NOT cover 3rd party solutions under its GDPR compliance features. As a responsible site administrator you yourself must ensure that any and all parts of your website are GDPR compliant, according to the law. 

= Can I use my existing theme? =

We've tested PeepSo with dozens of major themes. But since there are thousands of themes for WordPress, it is hard for us to test with all of them. Well, virtually impossible. The theme we recommend is [PeepSo Theme: Gecko](https://www.peepso.com/downloads/peepso-theme-gecko/) - it's a theme developed by the same team that makes PeepSo plugins and it can take your community to the next level. It is beyond anything 3rd party themes can offer with PeepSo. The theme is included for free in every single one of our bundles.

Having that said, you can check PeepSo with the 3rd party theme and if you find any problems, please report them and we'll fix whatever PeepSo might be breaking. However, if 3rd party theme itself is interfering with PeepSo's styles and overriding them in any way, there's nothing we'd be able to do - we can fix only what we've made ourselves. You'd have to contact the theme developers to get it fixed.

== Upgrade Notice ==

Upgrades can be performed in two ways. The automatic upgrade system lets you click a button on each plugin to upgrade to the latest available version. Alternatively, you can go to https://PeepSo.com/my-account/ and download the latest versions manually. Deactivate and delete old plugins, upload, and activate new plugins.

For automatic upgrades to work flawlessly, make sure to upgrade all the child plugins first and upgrade PeepSo last. You might need to reactivate the plugins too, once PeepSo is upgraded.
